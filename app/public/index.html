Open URL Shortner Service

What is this?

Open URL Shortner Service is an API which shortens the provided URL.

How to use this?

Send a post request to this endpoint with query paramter "url" containing the orignal URL; like

https://url-shortener-mubaidr.herokuapp.com/?url=https://www.google.com.pk;

and it will return an JSON object containing short URL:

{ url: "https://www.google.com.pk", short: "https://url-shortener-mubaidr.herokuapp.com/aBcCdD" }

How does it work?

Nodejs, Expressjs and monogoDB magic.
